# Triglav's Oracle

Triglav's Oracle is a small web app that turns any word into a three‑headed Wikipedia rabbit hole.

Live at https://triglav-wisdom-trails.lovable.app.

You enter a word, and the app builds a short "journey" of related Wikipedia pages, then lets you explore it through three aspects of the Slavic god Triglav:

- **Svarog's Forge** – creation, light, beauty, and high‑minded connections  
- **Perun's Strike** – direct, grounded, obvious links and mainstream topics  
- **Veles' Depths** – shadowy, sideways, and sometimes unsettling associations  

It is built with a Lovable template (Vite + React + shadcn/ui).
